# matrix

# Библиотека для работы с матрицами на C

Полноценная библиотека на языке C для выполнения операций с матрицами, разработанная в рамках учебного проекта. Реализует создание, арифметические операции, преобразования и продвинутые функции линейной алгебры с корректным управлением памятью и обработкой ошибок.

## ✨ Возможности

✅ **Управление памятью**
- Динамическое выделение через `calloc()` с обнулением ячеек
- Безопасное освобождение памяти через `remove_matrix()` для предотвращения утечек

✅ **Базовые операции**
- Создание и удаление матриц
- Сравнение матриц с учётом погрешности (`EPSILON = 1e-6`)
- Сложение, вычитание, умножение на скаляр
- Умножение матриц (скалярное произведение)

✅ **Продвинутая линейная алгебра**
- Транспонирование матрицы
- Вычисление определителя (рекурсивно, для любых размеров)
- Матрица алгебраических дополнений (кофакторов)
- Обратная матрица (для невырожденных квадратных матриц)

✅ **Надёжная обработка ошибок**
- Чёткие коды возврата: успех, общая ошибка, ошибка вычислений
- Проверка на `NULL` и валидация размеров на каждом шаге

✅ **Тестирование и качество кода**
- Модульные тесты с использованием фреймворка Check
- Отчёты о покрытии кода через `gcov`/`lcov`
- Строгие флаги компиляции: `-Wall -Wextra -Werror`

## 🛠 Требования

- **Компилятор**: GCC с поддержкой стандарта C11 (`gcc -std=c11`)
- **Инструменты сборки**: `make`, `ar`
- **Тестирование** (опционально): `libcheck` (пакет `check`)
- **Отчёты о покрытии** (опционально): `lcov`, `genhtml`



## 🚀 Установка и сборка

### 1. Клонируйте репозиторий

```bash
git clone git@github.com:your-username/matrix-lib.git
cd matrix-lib

2. Соберите статическую библиотеку

```bash
make all
# Результат: libmatrix.a

3. Очистка артефактов сборки

```bash
make clean

4. Пересборка + тесты одной командой

```bash
make rebuild



## Справочник по API

Основная структура

```c
typedef struct matrix_struct {
    double **matrix;  // Двумерный массив значений
    int rows;         // Количество строк
    int columns;      // Количество столбцов
} matrix_t;

 Коды ошибок

```c
#define OK 0           // Операция успешна
#define ERROR 1        // Некорректные аргументы / ошибка выделения памяти
#define CALC_ERROR 2   // Вычисление невозможно (несовпадение размеров, вырожденная матрица)

#define SUCCESS 1      // Для сравнения: матрицы равны
#define FAILURE 0      // Для сравнения: матрицы различаются

#define EPSILON 1e-6   // Допустимая погрешность при сравнении чисел с плавающей точкой
